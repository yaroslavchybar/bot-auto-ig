# Anti CLI

This is a Terminal User Interface (TUI) for the Anti Automation tool, built with [Ink](https://github.com/vadimdemedes/ink).

## Prerequisites

- Node.js (v16 or higher)
- npm

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the project:
   ```bash
   npm run build
   ```

## Usage

Run the CLI:

```bash
npm start
# or
node dist/cli.js
```

You can pass a name:

```bash
npm start -- --name=YourName
```

## Development

Run in watch mode:

```bash
npm run dev
```
